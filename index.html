customize

assembly instructions

  1. 1. customizing

    1. 1a. click on the rank to change card hierarchy
    2. 1b. click on "theme" section to change desired character

  2. 2. printing

    1. 2a. print a page for each rank until set is complete, total will be 13 sheets, cardstock is ideal
    2. 2b. from mobile devices: screenshot each set and airdrop to device with printer access
    3. 2c. from desktop device: file > print > select print destination > print

  3. 3. assembly: cut out along black outlines
